# 模板：requirements/facts.md

```
模板位置：vcddd/reference/templates/vcddd/requirements/facts.md
目标路径：docs/vcddd/requirements/facts.md
```

**定位**：经用户确认的结构化业务事实全貌。这是域设计的唯一输入。始终保持最新，不保留历史（历史在 changelog.md）。由模型翻译、用户确认后写入，不由用户直接撰写。

**写作要求**：
- 每条事实必须是已确认的，未确认的不写入
- 用业务语言描述，不出现技术实现词汇
- 域全景必须说明决策主权
- 协作关系必须标注方向和方式（命令/事件）
